# 说明

MatrixUtil类是一个功能丰富的工具类，专门用于处理矩阵操作。它提供了矩阵验证功能，确保矩阵的合法性和正确性。此外，该类支持矩阵的加法、减法和乘法运算，能够高效地执行这些常见的数学操作。MatrixUtil类还具备行反转功能，可以轻松地对矩阵中的行进行反转操作。这些功能使得MatrixUtil类在处理矩阵相关任务时非常实用和高效。
