# 说明

该AES加密解密类实现了高级加密标准（AES）的核心功能，包括密钥扩展、字节替换、行移位和列混淆等关键操作。该类专门设计用于处理128位数据的加密和解密，确保数据安全性和完整性。通过精确执行AES算法的各个步骤，该类能够有效地保护敏感信息，满足现代加密需求。
