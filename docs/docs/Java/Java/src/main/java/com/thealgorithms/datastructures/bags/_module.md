# 说明

Bag类实现了Iterable接口，具备添加元素、检查元素存在性以及遍历集合的功能。该类允许用户通过迭代器访问集合中的元素，支持动态添加新元素，并提供方法用于检查特定元素是否存在于集合中。Bag类的设计旨在提供灵活且高效的集合操作，适用于需要频繁添加和遍历元素的场景。
